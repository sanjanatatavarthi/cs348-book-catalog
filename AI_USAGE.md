# AI Usage

I used ChatGPT as an AI assistant while building this project.

## What AI helped with
- Brainstorming a simple project topic that would satisfy the CS348 project requirements
- Generating and refining Flask route structure for the CRUD and report pages
- Suggesting SQL queries for inserting, updating, deleting, and filtering books
- Suggesting input validation ideas and transaction structure for Stage 3

## How I verified or modified the output
- I read through the generated code and changed it to fit my project design
- I ran the application locally and tested add, edit, delete, and report features
- I checked that the SQL queries returned the expected results in the app
- I confirmed that dropdown values were loaded from the database instead of being hard-coded
- I reviewed the security, index, and transaction explanations against official documentation where needed

## Responsibility
I understand the final project is my own work. I reviewed the code, tested it locally, and made sure I can explain the database design, queries, indexes, SQL injection protection, and transaction choices used in this project.
